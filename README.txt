●Android用アプリ『CallingPlayerDiscordEdition』の説明

　　『CallingPlayerDiscordEdition』(*以降 当アプリ と呼称)はキャラクターと、通話を行っているというシチュエーションのASMR音声作品をより、臨場感のある形でお楽しみいただくことを目的としたAndroid用アプリケーションです。


　　起動後表示される初期画面では、通話相手のお名前を入力していただき"START"ボタンを押すことで通話画面を模した画面に遷移します。遷移先の画面の下部では左から順に、"音声を5秒戻す"、"音声の再生,停止"、"音声を5秒スキップ"、"プレイヤーを閉じる"といった機能になっています。



●Android用アプリ『CallingPlayerDiscordEdition』の注意点

　　開発者は当アプリにて発生した問題についての対応を行うことができないので、自己責任の上ご利用ください。また、当アプリを再配布することはご遠慮ください。


　　当アプリはセキュリティの観点でのテストを行なっていないので、必ず端末をネットワークから切断した状態で使用するようお願い申し上げます。


　　当アプリを使用後、端末の動作が遅くなってしまう不具合が確認されています。その場合は当アプリを終了後、端末の再起動を行うことで症状を回復できます。


　　当アプリはAndroidバージョン9.0での使用を想定したアプリケーションです。9.0以降のバージョンでは非推奨とされるメソッドを使用している箇所があるので、バージョン10.0以降では使用しないようにお願いします。また、テスト環境の不足からバージョン8.1以前の動作を確認していないので、動作する保証はありません。
